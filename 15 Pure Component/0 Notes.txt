2. Pure Component in React
React provides React.PureComponent (a class-based feature).
It is similar to React.Component, but it automatically implements shouldComponentUpdate() with a shallow comparison of props and state.

ğŸ‘‰ That means a PureComponent re-renders only if props or state have changed.
This can improve performance for components with unchanged props/state.

import React, { PureComponent } from "react";
class Counter extends PureComponent {
  render() {
    console.log("Rendering Counter...");
    return <h2>Count: {this.props.count}</h2>;
  }
}
export default Counter;

If parent re-renders without changing count, this PureComponent will not re-render, saving performance.
